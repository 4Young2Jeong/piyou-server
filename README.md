### **FrontEnd**

- `Unity`
- `Vue.js`
- `HTML`
- `CSS`
- `JavaScript`

- `Java 11`
- `Spring boot`
- `Spring Data JPA`
- `MySQL`
- `AWS EC2`

- `JIRA`
- `Notion`
